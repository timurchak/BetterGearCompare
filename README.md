# BetterGearCompare

BetterGearCompare is a lightweight World of Warcraft addon for retail that compares items by weighted stats.

The addon adds a short comparison block to item tooltips and can show an upgrade arrow on bag items.

Main features:
- weighted stat comparison with custom stat values
- separate profiles per specialization
- item tooltip summary with better, worse, or equal result in percent
- weapon comparison rules based on class and specialization
- optional upgrade arrow behavior that can consider item level

The addon is designed as a simpler alternative to larger gear comparison addons when only tooltip comparison and upgrade indication are needed.
